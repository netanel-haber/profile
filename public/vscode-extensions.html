Awesome VSCode Extensions

Builtin features you should know
  1. Fuzzy symbol search (v1.89+)
  2. New auto-indent settings
  3. Timeline for local history
  4. Check existing features before installing an extension
Essentials that are almost part of the editor
  1. Remote Development -Connect to remote envs
General niceties
  1. Highlight Words -Track symbols easily
  2. Error Lens -Inline error messaging
  3. Bookmarks -JetBrains-like
  4. Search Editor Apply Changes -Emacs-style compile mode
  5. Color Highlight -Highlight color strings, mostly css work
Extensions only needed for some workflows
  1. Kubernetes -Best K8s UI
  2. Rainbow CSV -CSV/TSV highlighting
  3. Copy Python Path -Convenient path copying
  4. Styled Components -Only if using styled-components
